# Security

If you discover a security vulnerability, please report it responsibly.

**Do not** open a public issue. Instead, report it via one of the following:

- **GitHub private vulnerability reporting:** Go to the repository **Security** tab → **Report a vulnerability**. This creates a private advisory and allows the maintainers to coordinate a fix and disclosure with you.
- If the repository has **Dependabot** or **Security policy** configured, you can use the contact option shown there.

Include a clear description of the issue and steps to reproduce. We will respond and work on a fix as appropriate.
