# Contributing to QuizParts

Thanks for your interest in contributing.

## Setup

1. Fork and clone the repo.
2. Run `yarn install`, `yarn build`, and `yarn dev` to confirm everything works.
3. Create a branch for your change.

## Making changes

- Follow the existing code style (ESLint + Prettier).
- Add or update tests as needed.
- For publishable package changes, add a [changeset](.changeset/README.md): `yarn changeset`.

## Pull requests

- Open a PR against `main`. CI will run lint, typecheck, tests, and build.
- Keep PRs focused; link related issues if applicable.

## Questions

Open a GitHub issue for discussion.
